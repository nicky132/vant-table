✨ 功能演示
VantTable 提供了丰富的表格功能，包括数据展示、选择、排序、过滤等。以下是主要特性：

🎯 选择功能

支持单选、多选模式，选择过滤，最大选择限制等高级功能

📌 固定列

支持左右固定列，自动阴影效果，完美的视觉体验

🔄 排序过滤

支持多列排序和自定义过滤，VTable风格的现代化交互

📱 响应式

完美适配移动端和桌面端，自适应各种屏幕尺寸

🎨 自定义渲染

支持单元格自定义渲染，灵活的内容展示方式

⚡ 高性能

支持大量数据渲染，虚拟滚动优化，流畅的用户体验

🚀 快速开始

安装:

npm install @nicky132/vant-table

使用:

import { VantTable } from '@nicky132/vant-table'
import '@nicky132/vant-table/dist/index.css'

// 在组件中使用
<VantTable
  :headers="headers"
  :data="data"
  :selectable="true"
  select-mode="checkbox"
/>
